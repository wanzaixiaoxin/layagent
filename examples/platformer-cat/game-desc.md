# 游戏描述：忍者猫咪跑酷

## 一句话描述
玩家控制一只忍者猫咪在森林中奔跑跳跃，躲避障碍物，收集金币。

## 核心玩法
- 自动向右奔跑（横版卷轴）
- 点击屏幕跳跃
- 空中再次点击进行二段跳
- 躲避障碍物（木箱、陷阱）
- 收集金币和能量道具

## 视觉风格
- 卡通风格，Q版角色
- 森林主题背景（远景/中景/近景三层视差）
- 色彩鲜艳明快

## 角色设定
- 主角：橘色忍者猫，穿黑色忍者服，大眼睛

## 道具
- 金币：金色，旋转动画
- 加速道具：闪电图标，使用后速度提升3秒
- 护盾道具：蓝色气泡，抵挡一次伤害

## UI 需求
- 开始按钮：绿色圆形，带播放图标
- 暂停按钮：右上角，两条竖线
- 分数显示：右上角，金色数字
- 生命显示：左上角，猫爪图标 x 3

## 目标平台
- H5 浏览器
- 微信小游戏
